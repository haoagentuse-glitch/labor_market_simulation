## 1. 核心提問

在什麼條件下,這個經濟收斂到「全面自動化」「混合均衡」「人類溢價經濟」——終局是參數的函數,還是被模型結構預先寫死的定理?

## 2. 立場與機制

### 2.1 診斷:單調終局從何而來(取代 v1–v4 的隱含假設)

四個結構元素共同鎖死終局:F 單調無界、(v1–v3 的)AI 成本單調趨零、保護項隨成熟度消失、任務空間封閉。spec v4 的 `c` 打開了一維(σ_max 成長),但 L_0.1.0 已證 c 救不了中低端——終局仍由 r 單向決定。**開放終局 = 給 AI 端補上缺失的負回饋,讓吸引子數量成為參數的函數。**

### 2.2 機制一:F 天花板(最小開放化,→ L_0.3.0)

```
dF/dt = r·(1 − F/F_ceiling)        # ceiling→∞ 退化回現行線性 r
```

`F_ceiling` 兩種用法:(a) **掃描軸**——相圖從 (r,g) 升為 (r,g,ceiling),看是否出現第三體制「前沿停滯下的人類溢價區」;(b) **隨機變數**——agent 與研究者都不知道真值,只能從 F 的減速中推斷,輸出從點預測變成**體制的貝葉斯後驗**(工程上=多種子 × ceiling 抽樣)。天花板的現實對應:算力/能源約束、資料耗盡、Polanyi 硬牆。

### 2.3 機制二:勞動所得份額(近零成本指標,即刻可加)

`labor_share = Σ V(σ_人類任務) / Σ V(全部任務)`——現有過程指標「AI 替代率」的補集。它讓 **Engels' Pause 型體制**(就業率穩、份額崩一整代)變得可見;現有就業率+Gini 的指標組合看不見這種歷史上真實出現過的狀態。

### 2.4 機制三:世代結構與主比值

Worker 有職涯長度 W(退休、新世代帶當期信念進場)。主比值:

```
圈周期/世代長度 = (F 掃過在職能力帶的時間) / W
```

比值 >1:調整靠世代交替(農業模式,近無痛);<1:同代人生涯中段被迫換圈(Engels' Pause 模式)。預期它取代 r 單獨值,成為「平穩 vs 陣痛」的真正分界。

### 2.5 機制四:編碼化回饋(成對對照)

```
F 在技能區 s 的增速 += κ · retraining_inflow(s)    # 再訓練湧入=為自動化備妥語料
```

與待加清單的「內生前沿」(部署驅動)同族但方向相反:一個是使用餵養、一個是**逃逸行為本身餵養**。必須成對跑對照,否則無法歸因。這是螺旋史觀「避難所自我掏空」不變量的機制化,也是 LLM 時代的字面事實。

### 2.6 驗收策略:歷史校準點(分階段)

| 校準點 | stylized 形狀 | 需要的機制 | 階段 |
|---|---|---|---|
| Engels' Pause | 就業穩 + labor_share 崩 | §2.3 | L_0.3.x 即可 |
| 農業轉型 | 慢 r + 世代交替 → 無痛重配置 | §2.4 | 世代後 |
| Bessen 倒 U | 自動化初期就業升、飽和後降 | 需求彈性隨飽和遞減(併入 spec 需求迴路設計) | 需求迴路後 |
| 中介職業弧線 | 近前沿帶(σ∈[F, F+δ])人口份額先升後降 | 現有引擎 | 即刻可量 |

### 2.7 遠期邊界(明列,不現在做)

**所有權分佈/馬邊界**:需先有分租規則(worker 拿 V(σ) 的份額、其餘歸資本——`30_exp_L0.1.0` §五D),才能讓「AI 任務價值歸誰」有意義,再談 UBI 融資與馬邊界那條線。**維度出生**(螺旋旋轉的機制化):1D σ 裝不下「新維度」,排在 concept v4 的 λν 原型之後,屆時另開 concept v6。

## 3. 可證偽假設

- **H1(終局開放)**:加入 F_ceiling 後,(r, g, ceiling) 空間出現 **≥3 種終局吸引子**(崩潰/重配置/前沿停滯人類溢價)。*推翻*:任何有限 ceiling 下末段行為仍只有既有二分 → 「單調終局」不是 F 無界造成的,診斷 §2.1 錯。
  → **【L_0.3.0 驗證:成立】** 兩張相圖均出現 collapse/reallocation/human_premium 三體制(`30_exp_L0.3.0` §3)。
- **H2(Engels' Pause 體制存在)**:存在參數區使末段就業率 ≥0.7 且 labor_share 較峰值下降 ≥15 個百分點。*推翻*:全參數空間就業與份額同向變動。
  → **【L_0.3.0 驗證:定性成立、定量待補】** 就業 0.97 而 labor_share 跌 14.0pp,方向確立;但最大跌幅差 1pp 未達 15pp 門檻,L_0.3.1 細掃低 ceiling 補。
- **H3(主比值定律)**:世代結構加入後,「平穩/陣痛」分界由圈周期/世代長度比值主宰。*推翻*:分界仍是 r≈常數的垂直線(與 W 無關)。
- **H4(編碼化自限)**:開啟 §2.5 後,再訓練聚集的技能區 F 增速上升、重配置區在相圖上收縮。*推翻*:κ 開關對相圖無可測影響。
