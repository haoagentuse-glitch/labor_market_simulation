## 1. 核心提問

描述「一個任務會不會、以及**多快**被 AI 取代」,最小需要幾個潛變數——兩個夠不夠?

## 2. 立場與機制

### 2.1 兩軸定義

- **λ 可表徵性**:任務的相關狀態有多少能被機器攝取與操作。高 λ=活在位元裡;低 λ=活在原子與默會裡。管「AI 能不能碰」。
- **ν 回饋密度(訊噪比)**:每單位時間可得的廉價正確訊號有多密。高 ν=編譯過/對答案(RL、self-play 可用);低 ν=好壞主觀、長期見真章。管「AI 學多快」。
  (ν 定義為回饋密度而非「可驗證性」,是為了解除它對 λ 的概念依賴,讓兩軸接近正交。)

例:下棋 λ高ν極高;法官 λ高ν低;水電工 λ低ν中;資料輸入 λ高ν高。

### 2.2 因果鏈(σ 是下游)

```
Task → (λ, ν) → AI 學習速度 → σ = 前沿距離(衍生) → 取代與否
```

σ 不是幾何距離,是**動力學誘導的到達時間** `T(λ,ν) = 1/(ν·g(λ))`:兩任務幾何等距,高 ν 者先被吃。

### 2.3 最小前沿律與三區

前沿是 (λ,ν) 上的場,演化律 `∂F/∂t ∝ ν·g(λ)`,且 `g(0)=0`(不可表徵者永久受保護)。速度場自然切出三區:高λ高ν=爆速被吃;高λ低ν=卡在「判斷層」;低λ=卡在「具身層」。**雙峰(匠人峰/判斷峰)必須從速度場長出,不得寫死。**

### 2.4 聚合=競爭,不選規則

職業=任務分布 `p(λ,ν)`;AI 在 `F(λ,ν,t) >` 門檻處逐任務接管;「職業消失」=仍在前沿上方的任務價值份額——是**結果**,不是聚合規則。這一步同時解掉兩層偷渡風險:任務→職業怎麼合成、(λ,ν)→取代怎麼合成,都不需要手選 min/加權。

### 2.5 底層合格判準(六條全過才算底層)

1 能生成四維可觀測量;2 能映射真實職業(潛變數由 manifest 反推,非直接打分,評分需通過再現性門檻如 ICC);3 能定義前沿演化;4 雙峰自然浮現;5 滿足前四條下維度最少(**最小生成基底原則**:可被更低維同生成力表示取代者,不是底層);6 潛變數可辨識。

### 2.6 研究定位(哪裡才真的新)

軸本身不新——λ≈曝光文獻的 data abundance+tacit knowledge,ν≈performance variance+efficiency gap(Felten AIOE;Eloundou 2023;arXiv:2510.13369)。真正新的是三件方法論:**靜態分數→動態生成**(前沿律+競爭聚合,極化從動力學長出)、**最小潛變數綱領**、**幾何 vs 機制對照實驗紀律**。Walo (2023) 證明 routine 測量彼此不一致——λν 若當指數打分必踩同坑,當潛變數反推是我們與曝光文獻的差異點。O*NET 已有 19,000 任務的近似打分,可借來實例化 `p(λ,ν)`。

## 3. 可證偽假設

- **H1(最小性)**:(λ,ν)+各向異性前沿擬合真實職業軌跡時**不需要第三軸**。*推翻*:需要 level/當責等第三維才能解釋變異 → 兩軸本體失敗。
- **H2(機制非幾何)**:雙峰來自速度場,不是可行域邊界形狀。*檢驗*:同一 `p(λ,ν)` 跑**各向同性前沿**(處處等速)對照——雙峰消失=動力學所生(過關);雙峰仍在=幾何假象(退回重想)。此實驗不做,等於沒驗證。
- **H3(軌跡驗收)**:模型應跑出三條真實軌跡——放射科(中段被吃、剩「責任殼」)、法官(長期卡低 ν)、水電工(受 λ 限幾乎不動)。*推翻*:任一條系統性跑錯。

## 4. 未決問題

1. **λ⊥ν 的耦合可能藏在速度場**(如 ∝min(λ,ν) 或 λ·ν),而非座標。了結方式:兩種速度場並列擬合 H3 軌跡,看何者需要更少補丁。
2. **ν 應拆 ν_intrinsic(任務本質)vs ν_data(資料生態,內生時變)**——後者會讓空間自己變形(低 ν 區隨資料累積變高 ν)。v1 原型先凍結為靜態,拆分排在原型驗證後。
3. **當責是不是能力軸外的「社會地板」**:`ν·g(λ)` 動力學下低 ν 只是慢、不是永不——模型會說法官終將被吃。若相信當責是社會性永久地板(法官不是 AI 不夠強,是社會不許),需另加地板項。這是設計抉擇,必須挑明,不可默默決定。arXiv:2604.04464(風險而非能力設限)是獨立佐證。
4. **量測再現性**:λ、ν 的 manifest 代理(可遠端化比例、廉價正確訊號存在與否)要先過評分者一致性門檻,一致性太低=軸尚未可操作化。

## 5. 交棒

**給 spec(下一版原型)**——closure-first 最小實作(約 50–80 行):

```
狀態:任務分布 p(λ,ν)(固定) + 前沿場 F(λ,ν,t)
律:  ∂F/∂t ∝ ν·g(λ)     # g(0)=0
接管:F > 門檻 → 任務歸 AI
讀數:存活價值份額、(λ,ν) 上的人類佔據雙峰、H3 三職業軌跡
對照:同一 p(λ,ν) 跑各向同性前沿(H2)
紀律:ν 凍結靜態、堅守 2D(不加 level)、先跑對照再信雙峰
```
